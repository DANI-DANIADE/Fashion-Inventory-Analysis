# Fashion-Inventory-Analysis
Fashion Inventory Analysis aims to improve inventory efficiency by identifying overstocked and underperforming products, optimizing stock allocation, and supporting data-driven decisions to reduce costs and increase sales.

# Dataset

This project uses a synthetic dataset designed to simulate real-world fashion retail inventory and sales data, including product categories, stock levels, and transaction patterns.
The dataset is suitable for demonstrating inventory analysis, stock optimization, and product performance evaluation.

# Tools

1. Excel (Data Cleaning)
2. SQL (Data Analysis)
3. Tableau (Visualization)

# Data Process

1. Cleaned raw inventory and sales data using Excel (handling missing values, formatting)
2. Analyzed stock levels, sales performance, and product turnover using SQL
3. Built an interactive dashboard in Tableau to visualize key inventory insights

# Key Insights

1. Several products show high stock levels but low sales, indicating overstock risk and inefficient inventory allocation
2. A small number of products contribute to the majority of sales, suggesting a Pareto distribution in product performance
3. Certain categories demonstrate faster stock turnover, highlighting opportunities to prioritize high-demand items
4. Slow-moving products increase holding costs and may require discounting or promotion strategies

# Executive Summary
The analysis reveals a mismatch between inventory levels and product demand, with significant capital tied up in slow-moving items. Optimizing stock allocation and focusing on high-performing products can improve inventory efficiency and increase overall profitability.
